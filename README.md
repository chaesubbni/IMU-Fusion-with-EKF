## 프로젝트 개요

본 프로젝트는 IMU 센서(자이로 + 가속도) 데이터를 확장칼만필터(EKF)로 융합해 정밀한 Roll/Pitch를 추정하고, 필터 파라미터까지 최적화 해 우리가 요구하는 환경에서도 안정적인 자세를 추정하도록 만들었습니다.
여기서, 확장칼만필터 및 필터 파라미터 최적화 알고리즘은 직접 구현했습니다. (외부 라이브러리 X)

여러 폴더들이 있지만 큰 틀은 단기적으론 자이로 센서로 예측하고 이때 발생하는 편차를 가속도 센서로 보정해 안정적인 자세 추정을 수행합니다.
또한, 상태변수에 Roll, Pitch만 넣는 것이 아닌 bias까지 넣어 편차까지 추정하도록 해 더 안정적인 예측값을 생성하도록 구현했습니다.


## 시연 영상

수평 상태에서 Roll, Pitch 오차가 0.0x도 이내로 매우 정밀합니다. 특히 센서를 회전시킨 후 다시 수평으로 복귀했을 때에도 누적 오차(Drift)가 발생하지 않으며, 소수점 첫째 자리까지 정확하게 0.0도를 유지하는 우수한 복원력을 보여줍니다.

## Twiddle 알고리즘 & 동적 환경 반영

- Twiddle 알고리즘은 PID 제어나 칼만 필터 등의 파라미터 최적화에 자주 사용되는 방식으로, 초기값을 기준으로 오차를 측정한 뒤, 성능이 개선되면 해당 파라미터를 더 크게(×1.1) 조정하고, 그렇지 않으면 반대 방향(-2×)으로 시도하는 방식입니다. 
- 만약 반대 방향에서도 성능 향상이 없다면 조정 폭을 줄여(×0.9) 탐색 범위를 좁혀 나갑니다.
- 결국 각 파라미터마다 최적값을 찾기 위해 ±k 범위를 반복적으로 탐색하며 오차를 최소화해가는 알고리즘.

- 또한, 시스템의 Q와 R을 보다 정밀하게 설정하기 위해, 정지 상태에서의 센서 노이즈, 움직인 후에 발생하는 동적 노이즈, 그리고 다시 원래 위치로 복귀했을 때의 편향(bias)을 반영한 노이즈까지 모두 고려해 총 오차(error)를 계산했습니다.
- 이러한 방식은 단순한 정적 오차 분석보다 실제 동작 환경을 반영한 종합적인 오차 측정이 가능하므로, 보다 현실적이고 신뢰도 높은 Q와 R 설정에 유리하다고 판단함.

## 최종 정확도
- 실제 동적 환경을 반영한 테스트와 Twiddle 알고리즘을 거쳐 도출한 최적의 파라미터로 Q, R 행렬을 업데이트한 결과, 최종 자세 추정 오차를 0.01° ~ 0.03° 수준으로 최소화했습니다.

## 느낀점

"칼만필터는 어렵지 않아"라는 책을 3번 회독하고 관련 유튜브 영상들을 보며 외부 라이브러리를 사용하지 않고 직접 확장칼만필터를 구현해봤습니다.
물론, 초반엔 테스트 할 환경을 고려하지 않아 기울기 추정의 정확도가 매우 낮았지만 수많은 테스트와 최적의 파라미터 값을 찾아 정확도를 향상시켰을 때 매우 뿌듯했습니다.
이런 칼만필터가 여러 분야에서 많이 쓰이는 걸로 알고 있는데 뭔가 PID 제어처럼 새로운 기술을 얻은 느낌이었고 생각보다 어렵지 않다는 걸 느꼈습니다.
만약 여러분들도 "칼만필터가 어려운 거 같아.." 라고 생각이 드신다면 이 "칼만필터는 어렵지 않아"를 꼭 추천합니다!!!
